# Uber Ride Analysis 🚖

## 📌 Project Overview
This project analyzes Uber ride data to uncover insights about ride patterns, demand trends, and customer behavior.  
The analysis is built in **Power BI** using interactive dashboards for visualization.

## 🎯 Objectives
- Identify peak ride hours and days
- Understand demand distribution across locations
- Analyze fare trends and ride distances
- Provide actionable insights for business decisions

## 🛠️ Tools & Technologies
- **Power BI** for dashboard creation
- **Data Cleaning & Transformation** using Power Query
- **Visualization** with charts, maps, and KPIs

## 📊 Key Features
- Interactive dashboards with filters
- Trend analysis of rides over time
- Geographic insights using map visuals
- Fare and distance correlation analysis
